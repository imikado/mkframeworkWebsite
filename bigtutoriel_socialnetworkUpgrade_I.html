I Préambule

I.A Introduction

Ce tutoriel s'appuie sur le précédent : Comment créer son propre réseau social
Il vise à améliorer cette ébauche pour apprendre à faire évoluer un projet.

Ce tutoriel est écrit au fur et à mesure des demandes que vous pouvez faire sur forum de developpez.com

I.B Créons nos deux bases de données


  
CREATE TABLE Likes (
id INT NULL AUTO_INCREMENT,
user_id INT NULL,
post_id INT NULL,

PRIMARY KEY id (id)
);
CREATE TABLE Comments (
id INT NULL AUTO_INCREMENT,
user_id INT NOT NULL,
post_id INT NULL,
body VARCHAR(250) NULL,
date DATE NULL,
time TIME NULL,
PRIMARY KEY id (id)
);
