You can install vith composer:

Composer is a project which provide you the way to manage dependencies: you can create a file to write what libraries you need and what version.

Create a file composer.json on your root project with this content:
php >= 7.2

  
{
   "require":{
       "mkframework/mkframework-lib":"5.1.0"
   }
}
    

php >= 5.2 < 7.2

  
{
   "require":{
       "mkframework/mkframework-lib":"4.138.0"
   }
}
    

Make a directory "vendor" on the project root, the framework will be install inside
Then launch this line :

  
composer install
   

More information on composer here
