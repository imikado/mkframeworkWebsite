Test unitaires: Lancer les tests

Retour au menu

Introduction

Vos tests écrit, vous pouvez soit les lancer via la ligne de commande, votre IDE ou encore comme ici: le builder

Cliquez sur le bouton "Lancer tests unitaires"



Le premier lancement



Au premier lancement, vous voyez cet écran qui vous demande de télécharger le framework de test unitaire: phpunit.
Rendez-vous à l'adresse https://phpunit.de/ puis téléchargez le fichier .phar à la racine du répertoire de votre application

Une fois fait, réactualiser cette page



Comme vous pouvez le remarquer, le champ "phpunit" s'est remplit avec le nom de votre fichier .phar

Si vous avez l'erreur suivante

  
sh: 1: data/genere/aTestUnit//phpunit-4.8.27.phar: Permission denied
    


Ajoutez le droit d'execution sur ce fichier .phar puis relancer avec le bouton "lancer"

Vous aurez la sortie suivante en cas de succès


Et celle-ci en cas d'erreur
